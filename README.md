# GigCrowd

A social network for tracking concerts and live events.

## 🎯 About

GigCrowd is a social platform where users can:
- Discover concerts worldwide
- Mark events as "going", "maybe", or "went"
- Follow other users
- See a social activity feed
- Post photos/videos from events
- Track personal concert history
